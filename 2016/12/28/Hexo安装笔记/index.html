Hexo安装笔记

发表于 2016-12-28

Hexo 是一个简单地、轻量地、基于Node的一个静态博客框架，可以方便的生成静态网页托管在github和Heroku上，引用Hexo作者 @tommy351 的话：
快速、简单且功能强大的 Node.js 博客框架。A fast, simple & powerful blog framework, powered by Node.js.

下面的教程比较长，但是不需要被这个吓退，小白可以按照步骤一步步操作完成自己的站点搭建。建好以后就是一劳永逸的事了。

以下安装是在mac上进行….

##1.本地安装
安装git。（mac自带不需要认为安装）

安装nodejs。

安装hexo：

这里假设你的个人域名是my.me

  • 安装,参考官网说明

至此hexo就安装完成，在浏览器中打开 http://localhost:4000 即可看到效果

  • 配置github,在配置文件里面配置自己的github的url

##2.github配置

###申请一个github账号，并创建一个项目

  • 要注意项目名称，格式是：

$ mkdir devzhao.com
$ cd devzhao.com
…

$ cd /home/user/admin/my.me
$ echo “# test.github.io” >> README.md
$ git init
$ git add README.md
$ git commit -m “first commit”
$ git remote add origin https://github.com/你的用户名/你的用户名.github.io.git
$ git push -u origin master

hexo g

hexo s –debug

hexo d

$ hexo server
$ hexo s –debug #启动 Hexo 本地站点，并开启调试模式

$ hexo new “输入一个名字作为你的md文件名”

$ cd /home/user/admin/my.me
$ hexo clean #简写为 hexo c
$ hexo generate #简写为 hexo g
$ hexo deploy #简写为 hexo d
