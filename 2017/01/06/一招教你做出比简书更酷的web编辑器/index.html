Jan 6, 2017 | Hits

在简书的不知道哪个版本的编辑器出来之前，在网上写图文文章都要经历蛋疼的图片上传过程。终于有一天，这个问题被简书的“直接粘贴图片”功能终结。而终于有一天我发现了一个叫做summernote的编辑器，通过它和七牛云服务实现了简书的“直接粘贴图片”的功能。
